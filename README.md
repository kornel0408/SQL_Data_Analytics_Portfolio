# SQL Portfolio

Welcome to my SQL Portfolio repository.

This repository contains end-to-end SQL data analytics projects built using real-world business datasets. Each project focuses on solving business problems using SQL concepts such as joins, aggregations, subqueries, CTEs, window functions, views, and advanced filtering techniques.

The goal of this repository is to demonstrate my SQL skills for data analytics, data cleaning, business analysis, and insight generation.

## Tools Used

* MySQL Workbench
* SQL
* CSV Datasets

## Projects Included

### 1. E-Commerce Analysis

This project focuses on analyzing customer orders, sales performance, product categories, payment trends, and delivery performance.

Key topics covered:

* Customer analysis
* Revenue analysis
* Top-selling products
* Delivery delays
* Order trends
* Business insights

### 2. Sales Analysis

This project focuses on identifying sales trends, top-performing products, monthly revenue, regional performance, and profit-related insights.

Key topics covered:

* KPI analysis
* Monthly sales trends
* Product performance
* Revenue growth
* Regional analysis
* Advanced SQL queries

### 3. Customer Behavior Analysis

This project focuses on understanding customer purchasing behavior, repeat customers, segmentation, and retention analysis.

Key topics covered:

* Customer segmentation
* Repeat purchase analysis
* Retention analysis
* Customer lifetime value
* Purchase frequency
* Advanced filtering and ranking

### 4. HR Analytics

This project focuses on employee-related insights such as attrition, department performance, salary analysis, and employee trends.

Key topics covered:

* Employee attrition
* Salary trends
* Department analysis
* Hiring trends
* Employee performance
* Business recommendations

### 5. Banking Analysis

This project focuses on analyzing loan applications, customer demographics, approval rates, and repayment behavior.

Key topics covered:

* Loan approval analysis
* Customer demographics
* Risk analysis
* Default trends
* Repayment performance
* Business insights
